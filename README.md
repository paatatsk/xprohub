# XProHub

**Real Work. Fair Pay. For Everyone.**

XProHub is a dual-role marketplace where every user can both post a job and take a job — no permanent roles. Customers describe what they need and set a budget; workers offer their skills and set their rates. Payment is held in escrow via Stripe and released when work is confirmed. A flat 10% platform fee, shown transparently. Built for individuals, in New York City.

---

## How This Was Built

XProHub was designed and built by **Paata Tskhadiashvili** as a solo founder in New York City, using an AI-assisted "orchestra model" of collaboration:

- **Paata** — founder and conductor. Every product decision, design judgment, and direction call. Hardware-verified every change on device before committing.
- **Claude (chat)** — strategist and reviewer. Strategy, framing, prompt drafting, decision architecture, honest critique.
- **Claude Code** — builder. Full-stack engineering, code reviews, database migrations, deployments.
- **Claude Design** — UI/UX designer. Mockups, design system, copy contracts, handoff packages.

The build spanned multiple Claude model generations (Opus 4.6 through 4.8) across sessions from March 2026 onward. The product vision, design taste, and every final call were the founder's; the AI tools served as strategy, engineering, and design assistance.
